# LoreGUI screenshots

UI captures for the website and user guides. Most are generated by the headless
visual harness (`frontend/scripts/visual-test.mjs`) — re-run it to refresh:

```sh
npm --prefix frontend run build
(cd frontend && npx vite preview --port 4173 &)
OUT_DIR=website/public/screenshots npm --prefix frontend run screenshots
```

The harness writes here by default. `native-*.png` are real native-binary captures
(`cargo tauri build` + xvfb), kept by hand.
